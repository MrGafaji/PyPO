Introduction

Welcome to the PyPO user manual. Here you can find several resources that will assist in using and/or developing PyPO. This homepage contains general information regarding installation instructions, the content of the user manual, contributor guidelines and contact information. After installing, the best place to start for new users would be the PyPO fundamentals tutorials, after which the rest of the tutorials can be studied. For contributor guidelines and general development instructions, please see the instructions for developers page.

Installation

The prerequisites for installing PyPO on any operating system are:

  • A working Python installation of version 3.8 or higher.
  • A pip installation.

Install From Source

Linux

First, install CMake:

pip install cmake

All other necessary prerequisites are fetched and installed upon installing PyPO. To install PyPO, clone the git repository:

git clone https://github.com/PyPO-dev/PyPO

Alternatively, download the .zip package from here and unpack in the preferred destination. Navigate to the cloned/unpacked repository and run:

pip install .

MacOS

Installing PyPO from source on MacOS proceeds largely along the same lines as for Linux. The only difference being that you will need to have the command line tools package installed:

xcode-select --install

Then, installation of PyPO proceeds in the same fashion as for Linux.

Windows

Install CMake through pip:

pip install cmake

Then, install Visual Studio Express. Choose the C++ desktop development package. Important is, is that the MSVC compiler is present

After this is installed, PyPO can be installed just like on Linux or MacOS.

Dev Install

In order to develop PyPO, we recommend to install PyPO in edit mode. See this section of the developers guidelines page for more info.

Using PyPO On GPU

In order to use the GPU accelerated libraries for PyPO, a working CUDA installation should be present. For Linux, this article describes nicely how to install CUDA through the apt repository.

For Windows, see the official instructions on the Nvidia website.

Unfortunately, Apple has stopped supporing Nvidia cards since 2015 and therefore, the CUDA libraries for PyPO cannot be installed on MacOS currently.

Optional Dependencies

PyPO contains several optional features such as a graphical user interface (GUI). For developers, an automated testing framework is present as well as a documentation generator. The prerequisites for these features are not installed through the regular install. Instead, they can be installed separately as needed.

To use the GUI, PySide2 and attrs needs to be installed:

pip install PySide2 attrs

The automated tests are run using the nose2 framework:

pip install nose2

Documentation in PyPO is generated using Doxygen. Please see their installation page for instructions regarding installation.

User Manual

Several tutorials have been developed in order to demonstrate the usage and functionality of PyPO:

  • the first group of tutorials are meant to show the fundamentals of PyPO and the design philosophy,
  • the second set of tutorials showcase the functionalities of the package,
  • the third and final set show how to use the GUI.

The software documentation for PyPO is split into two parts:

  • the first part forms a comprehensive API reference for all public methods of PyPO,
  • the second part forms a reference for all internal Python, C/C++ and CUDA code.
